# SCLS - Math
## What is SCLS ?
SCLS is an Aster System's set of libraries made to simplify development in C++, with a lot of useful and simple tools made to simplify the life of C++ developers.
## What is SCLS Math ?
SCLS Math "Carl" is the mathematical part of the SCLS library. It contains a lot of interesting tools made to use some abstract and non-abstract mathematics tools.
## Requirements
This library is made for C++ 20, but can be used in C++ 17.
It does not need any other requirements than the library SCLS Foundation "Leonhard".
The include path to the SCLS Foundation headers can be specified in a pre-processor directive : SCLS_FOUNDATION_PATH (it must be written before including SCLS Math).
